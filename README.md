# 校园展示网页 README

## 项目名称

Campus Display - 校园风采展示网页

## 项目简介

本项目为网页设计大赛参赛作品，采用纯原生HTML+CSS+JavaScript开发，无需搭建服务器、无需数据库，本地解压后即可直接在浏览器打开展示。网页以“校园风采”为核心主题，聚焦校园风景、校园地标、校园文创、校园生活四大板块，通过简洁清新的设计、流畅的交互效果，全方位展现校园的青春活力与独特魅力，适配电脑、手机等多终端，满足大赛展示与源码交付需求。

## 开发环境

- 开发语言：HTML5 + CSS3 + JavaScript（原生，无框架依赖）
- 开发工具：VS Code（推荐）、Notepad++ 等任意文本编辑器
- 运行环境：Chrome、Edge、Firefox 等主流浏览器（本地打开即可，无需服务器）

## 项目结构

文件夹结构清晰，便于查看与修改，符合大赛源码提交规范：
campus-display/          // 项目根目录（纯英文无空格，避免加载异常）
├─ index.html            // 主页面（入口文件，所有页面跳转核心）
├─ css/                  // 样式文件夹
│  ├─ style.css          // 全局样式（布局、配色、基础样式）
│  └─ responsive.css     // 响应式样式（适配手机、平板等终端）
├─ js/                   // 脚本文件夹
│  └─ main.js            // 核心脚本（轮播、悬浮、平滑滚动等交互效果）
├─ images/               // 图片文件夹
│  ├─ campus/            // 校园风景、地标图片
│  ├─ creative/          // 校园文创作品图片
│  └─ logo/              // 校园LOGO、图标图片
└─ README.md             // 项目说明文档（本文件）

## 功能介绍

1. 首页轮播：展示校园核心风景、地标全景图，自动轮播+手动切换，搭配淡入淡出动画，视觉效果清新自然。
2. 校园风采板块：分为校园风景、校园地标、校园文创三个子模块，采用卡片式布局，hover时有悬浮效果，点击可查看大图。
3. 校园生活板块：展示校园日常、活动剪影，以图文结合的方式呈现，排版简洁美观，突出青春气息。
4. 响应式适配：自动适配电脑、手机、平板等不同尺寸屏幕，保证在各类设备上都能正常显示，无布局错乱。
5. 交互效果：包含平滑滚动、导航栏滚动变色、图片懒加载、卡片悬浮等细节效果，提升用户体验，符合大赛评分要点。

## 使用说明（重要，适配大赛展示）

1. 获取源码：下载项目压缩包， 必须先解压整个压缩包（禁止直接在压缩包内双击打开index.html，会导致样式、图片加载失败）。
2. 打开网页：解压后，找到根目录下的 index.html 文件，双击即可在本地浏览器打开，无需其他操作。
3. 查看板块：通过顶部导航栏，可快速跳转至校园风景、地标、文创、生活等板块，体验完整功能。
4. 适配说明：若打开后无样式/图片，检查以下两点：① 已解压整个文件夹；② 文件夹名称为纯英文无空格、无特殊符号。

## 设计亮点（大赛加分项）

- 主题贴合：聚焦校园场景，素材易获取、内容接地气，传递青春正能量，符合评委审美偏好。
- 视觉设计：采用清新简约的配色（以校园代表色为基础），排版整齐，留白合理，视觉层次清晰。
- 交互细节：加入多种轻量化交互效果，无冗余代码，运行流畅，提升页面质感，突出前端设计能力。
- 易维护性：代码结构清晰，注释完整，样式与脚本分离，便于后续修改与优化，符合大赛源码规范。
- 兼容性强：适配主流浏览器，本地打开即可运行，无需依赖任何第三方工具或服务器，方便大赛展示与评委审核。

## 注意事项

- 请勿修改文件夹层级结构，否则会导致样式、图片、脚本加载失败。
- 图片素材均为校园相关，若需替换，可直接替换images文件夹内对应图片，保持文件名一致即可。
- 提交大赛时，需将整个项目文件夹压缩（根目录为纯英文），提交压缩包+源码，评委解压后即可正常查看。
- 若使用Chrome浏览器打开无样式，可切换至Edge或Firefox浏览器，或右键index.html选择“以Edge打开”。
